# Bread Machine Recipes

Recipes developed or adapted specifically for automatic bread machines.

These protocols should document:

- Machine model used for validation
- Ingredient loading order
- Course and crust settings
- Dough capacity and finished loaf size
- Any machine-specific adjustments
- Observations from each cook
